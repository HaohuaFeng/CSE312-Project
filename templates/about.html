About FaceNote
FaceNote is a powerful web application.

FaceNote has the function of Blog which clients can share their great moment with other. Right now, the blog section permits user to upload several types of multimedia, i.e. all kinds of images.

Another wonderful feature of FaceNote is it allows user to send a private message to another user online. When a new private message is received, there is a showy notice displayed without disturbing you.

The most brilliant section of FaceNote is the game section. User can enter the game page and invite all online users to start an interesting game. In the game page, players are listed and players can communicate to other. One user can draw on the board and other can guess what he/she is drawing.

Explore more functions of FaceNote, have a great time!
